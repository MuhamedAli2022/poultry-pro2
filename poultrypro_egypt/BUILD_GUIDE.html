🚀 خطوات بناء APK

الطريقة 1: الأسهل (Batch Script)

Step 1: افتح Batch File
اذهب إلى المجلد وابحث عن build_apk.bat
Step 2: انقر مرتين (Double Click)
سيفتح Windows CMD ويبدأ البناء
Step 3: انتظر (5-10 دقائق)
اترك الـ CMD مفتوح حتى الانتهاء
Step 4: اقبل الـ License
قد يطلب منك قبول Android SDK licenses

الطريقة 2: من Terminal

cd d:\VSCode-win32-x64-1.127.0\github\poultrypro_egypt flutter clean flutter pub get flutter build apk --release

الطريقة 3: APKs مقسمة (الأفضل)

flutter build apk --split-per-abi --release
⚠️ تنبيه: البناء قد يستغرق 5-15 دقيقة. لا تغلق الـ CMD أثناء البناء!

🔧 حل المشاكل الشائعة

❌ مشكلة: "Flutter not found"

تأكد من تثبيت Flutter وإضافته للـ PATH flutter --version

❌ مشكلة: "Android SDK not found"

android/local.properties يجب أن يحتوي على: sdk.dir=C:\Users\Elbeda F\AppData\Local\Android\sdk

❌ مشكلة: "java version"

تأكد من تثبيت Java 11 أو أعلى java -version

✅ الحل الشامل

flutter doctor flutter doctor --android-licenses flutter clean flutter pub get flutter build apk --release --verbose

📦 موقع الملف النهائي

بعد الانتهاء، ستجد APK هنا:
d:\VSCode-win32-x64-1.127.0\github\poultrypro_egypt\ build\ app\ outputs\ flutter-apk\ app-release.apk ✅

📱 تثبيت على الهاتف

الطريقة 1: عبر USB

فعّل USB Debugging على الهاتف
اتصل الهاتف بالـ PC عبر USB
شغّل: flutter install

الطريقة 2: انسخ الملف مباشرة

انسخ app-release.apk للهاتف
افتح ملف المدير على الهاتف
انقر على APK واختر "تثبيت"
اسمح بالتثبيت من مصادر غير معروفة
